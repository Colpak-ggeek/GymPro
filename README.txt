========================================
  GymPro — Система управления спортзалом
========================================

ЗАПУСК (3 шага):
----------------
1. Установите зависимости:
   pip install flask

2. Запустите приложение:
   python app.py

3. Откройте браузер:
   http://127.0.0.1:5000

ЧТО УМЕЕТ ПРИЛОЖЕНИЕ:
----------------------
Администратор:
  - Дашборд со статистикой
  - Создание пользователей (клиент / тренер / администратор)
  - Удаление пользователей
  - Создание тренировок (тренер, тип, день, время, зал, кол-во мест)
  - Удаление тренировок
  - Просмотр участников каждой тренировки
  - Назначение / отмена абонементов клиентам
  - Обзор всех абонементов

Клиент:
  - Самостоятельная регистрация через /register
  - Личный кабинет с данными профиля
  - Расписание занятий с фильтром по дням
  - Запись на тренировку и отмена записи
  - Оформление абонемента (Базовый / Стандарт / Премиум)
  - Заморозка / разморозка абонемента

Тренер:
  - Просмотр своих занятий
  - Список участников каждого занятия


ФАЙЛЫ:
------
  app.py           — весь код приложения (один файл)
  requirements.txt — зависимости
  gym.db           — база данных (создаётся автоматически при первом запуске)
